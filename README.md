# MoneyTrack

Aplicación web para control de gastos personales. Permite gestionar saldo, ingresos, gastos, categorías y metas de ahorro.

## Tecnologías
- HTML, CSS, JavaScript (sin frameworks)
- Datos en localStorage

## Uso
Abre `index.html` en tu navegador para usar la aplicación.

## Estructura
- **Inicio**: Dashboard con resumen
- **Saldo**: Configurar saldo inicial (efectivo y banco)
- **Ingresos/Gastos**: Registrar movimientos
- **Categorías**: Gestionar categorías de gastos
- **Metas**: Objetivos de ahorro con seguimiento
- **Administrar**: Ver y gestionar todos los datos
